# Student Performance Insight Dashboard (2013-2023)

## 📌 Project Overview
An end-to-end data analysis project visualizing 10 years of student performance data. The goal was to identify correlations between study habits, attendance, and CGPA growth across different demographics.

![Dashboard Preview](Images/screenshot.png)

## 🚀 Key Insights
* **The Attendance Paradox:** Male students showed 29% higher attendance but lower CGPA growth compared to female counterparts.
* **Weak Correlation:** A low R² of 0.15 indicates that study hours alone are a poor predictor of credits earned in this dataset.
* **Peak Intake:** Enrollment peaked significantly in 2020 before stabilizing.

## 🛠️ Tech Stack & Skills
* **Tool:** Power BI / Power Query
* **Data Modeling:** Star Schema (Fact and Dimension tables)
* **Analytical Techniques:** Correlation Analysis ($R^2$), Trend Forecasting, and Gender-Based Benchmarking.
* **Aesthetics:** Custom Dark/Gold professional theme.

## 📂 How to Use
1. Download the `.pbix` file from this repo.
2. Open in Power BI Desktop to explore the DAX measures and data transformations.
